Andrew Arcade

Andrew Arcade is a custom game console built with a Raspberry Pi 5 running Linux. Its main purpose is to play Unity games that I have created. Since the Raspberry Pi 5 uses an ARM processor, Unity games can't be compiled to run on it directly. To solve this, we use a tool called Box64, which lets us run Unity games and apps on the Raspberry Pi by emulating the required architecture.

Their work includes the horror title Hell Hotel, the fast-paced arcade game Oranyth, the Unity tool Juicy Player Controller, and the action game Sergent Wiskers' Brigade.
