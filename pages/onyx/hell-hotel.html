About Hell Hotel:

Hell Hotel is a first person horror game developed by ONYX Development using the Unity Game Engine.

The project was started late summer of 2023 and we have been working on it from time to time.

We do plan on eventually releasing the game to the public on steam (and possibly console) however we do not have a time release date yet.
