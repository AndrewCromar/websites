About Juicy Player Controller:

Juicy Player Controller is a unity first person all-in-one script to help you get started with first person movement.

We tried to make it as simple to understand as possible.

All you have to do is download the package and drag the prefab into your scene! It has default recomended setting so it works out of the box.

Features:

  • Walking.
  • Running.
  • Crouching.
  • Looking around.
  • Cursor hiding.
  • Fully custom controller tuning.
  • Built in default recommended settings.
  • Helpful error logging.
  • Debug variable in the inspector.
  • Epic logo.
