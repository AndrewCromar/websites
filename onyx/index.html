JUICY PLAYER CONTROLLER

A single c# script can have it all? Yes, Juicy Player Controller is a single c# script for Unity 3d projects to act as a base player controller for any first person or third person Unity 3d projects.


Juicy Player Controller is seperated into two flavors: orange and lemonade. The orange flavor is for first person and the lemonade flavor is for third person.


Features

  • Walking.
  • Running.
  • Crouching.
  • Looking around.
  • Cursor hiding.
  • Fully custom controller tuning.
  • Built in default recommended settings.
  • Helpful error logging.
  • Debug variable in the inspector.
  • Epic logo.
