Oranyth

About

The player moves around a colored dot to collect other dots of their same color. Every once and a while the players color will change. Collecting the correct color adds time while collecting the wrong color removes time. Time is always ticking down, survive as long as possible. Power-ups such as time freeze and rainbow (collect any color) can be collected when they popup.


The game supports keyboard (wasd or arrows) and controller (joysticks or d-pad). We might add mobile support soon.


You can play the game here.

Power-Ups

Time Freeze

The time freeze power-up stops the time from ticking down for five seconds.

Rainbow

The rainbow power-up allows the player to collect any color for five seconds.

Multi Point

The multi point power-up gives more than one point when collected.

Juicy Player Controller

About

A single c# script can have it all? Yes, Juicy Player Controller is a single c# script for Unity 3d projects to act as a base player controller for any first person or third person Unity 3d projects.


Juicy Player Controller is separated into two flavors: orange and lemonade. The orange flavor is for first person and the lemonade flavor is for third person.


Features

  • Walking.
  • Running.
  • Crouching.
  • Looking around.
  • Cursor hiding.
  • Fully custom controller tuning.
  • Built in default recommended settings.
  • Helpful error logging.
  • Debug variable in the inspector.
  • Epic logo.

If you are interested, you can learn more and download at GitHub.
