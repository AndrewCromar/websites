Andrew Arcade

Andrew Arcade is a custom game console running Linux on a Raspberry Pi 5. It is designed to run my own Unity games.

Version 2 is currently in development. This update focuses on moving from Linux to Android for better native support on ARM architecture. There is no set completion date for Version 2, but progress is being made slowly.

Eventually, I plan to create a separate website for Andrew Arcade. For now, this page serves as its home.
